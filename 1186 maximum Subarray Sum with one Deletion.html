1186 maximum Subarray Sum with one Deletion
Given an array of integers, return the maximum sum for a non-empty subarray (contiguous elements) with at most one element deletion. In other words, you want to choose a subarray and optionally delete one element from it so that there is still at least one element left and the sum of the remaining elements is maximum possible.
Note that the subarray needs to be non-empty after deleting one element.
 
Example 1:
Input: arr = [1,-2,0,3]
Output: 4
Explanation: Because we can choose [1, -2, 0, 3] and drop -2, thus the subarray [1, 0, 3] becomes the maximum value.
Example 2:
Input: arr = [1,-2,-2,3]
Output: 3
Explanation: We just choose [3] and it's the maximum sum.
Example 3:
Input: arr = [-1,-1,-1,-1]
Output: -1
Explanation: The final subarray needs to be non-empty. You can't choose [-1] and delete -1 from it, then get an empty subarray to make the sum equals to 0.
 
Constraints:
  • 1 <= arr.length <= 10^5
  • -10^4 <= arr[i] <= 10^4
we will solve this question in O(n) time using Kadanne's algorithm
calculate
maximum sum left  array   left[i] = max(left[i-1] + arr[i],arr[i]) 
maximum sum right          right[i] = max(right[i+1] + arr[i],arr[i])
ex :
arr  = [1,-2,0,3]
left = [1,-1,0,3]
right = [2,1,3,3]
let us say we want to delete ith element
then 
maxsum upto that index will be :
max sum of left array upto i-1 + max sum of right array  from i+1 ownwards
therefore 
maxSumUptoEachIndex = left[i-1] + right[i+1]       (i >(1 to n-2))
find max element from the above array that is the  answer
CODE:
int maximumSum(vector<int>& arr) {
       if(arr.size() == 1)
           return arr[0];
        int n = arr.size();
        int left[n];
        left[0] = arr[0];
        int right[n];
        right[n-1] = arr[n-1];
        int ansmax  = INT_MIN;
        for(int i=1;i<n;i++){
            left[i] = max(left[i-1]+arr[i],arr[i]);
            ansmax = max(ansmax,left[i]);
        }
        for(int i=n-2;i>=1;i--){
            right[i] = max(right[i+1] + arr[i],arr[i]);
            ansmax = max(ansmax,right[i]);
        }
        for(int i=0;i<n-2;i++){
            ansmax =  max(ansmax,left[i] + right[i+2]);
        }
     return ansmax;
    }
