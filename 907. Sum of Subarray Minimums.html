907. Sum of Subarray Minimums
Given an array of integers A, find the sum of min(B), where B ranges over every (contiguous) subarray of A.
Since the answer may be large, return the answer modulo 10^9 + 7.
 
Example 1:
Input: [3,1,2,4]Output: 17Explanation: Subarrays are [3], [1], [2], [4], [3,1], [1,2], [2,4], [3,1,2], [1,2,4], [3,1,2,4].
Minimums are 3, 1, 2, 4, 1, 1, 2, 1, 1, 1. Sum is 17.
 
Note:
  1. 1 <= A.length <= 30000
  2. 1 <= A[i] <= 30000
Intution : 
  1. We can have a O(n^2) approach, where we make two loops for i  and j
go through each subarray [i:j] and keep track of min for that subarray and keep adding it to final sum.
But here Array length can be at max : 10000
so n^2 solution may lead to TLE

  1. O(n) approach
We maintain two arrays left and right
where, left[i] means no of elements larger than A[i] in left
right[i] means no of elements larger than A[i] in right

Initially, we can initialize these two arrays by the intution that we assume for any index 'i' all elements in left and right are larger than this element
therefore, left[i] = i and right[i] = n - i - 1        //n is A.size()

Now, we maintain two stacks sl,sr for left and right subarrays

Let's Update our right array correctly now
for i 0 : n-1
{
while(sr is not empty and  A[sr.top()] > A[i])       // this means the element sr.top() is not smaller than an element i on it's right , so our initial assumption was wrong and we update it as follows
    { 
      right[sr.top()] = i - sr.top()-1;
     sr.pop();
    }
sr.push(i);
} 

//y we can update left array
for i n-1 : 0
{
while(sl is not empty and  A[sl.top()] >= A[i])       // this means the element sl.top() is not smaller than an element i on it's left , so our initial assumption was wrong and we update it as follows
    { 
      leftt[sl.top()] = sl.top()-1-i;
     sl.pop();
    }
sl.push(i);
}


after this, for any element i number of elemnts on its left are  : left[i] and on right are : right[i]
so total number of subarrays inclusing this number are : (left[i]+1)*(right[i]+1)    //+1 because there is one subarray where only this element is present

finally, sum += A[i]*(left[i]+1)*(right[i]+1)   // over all elements

int sumSubarrayMins(vector<int>& A) {
     int n = A.size();
     vector<int> larger_in_left(n,0),larger_in_right(n,0);      for(int i=0;i<n;i++){
         larger_in_left[i] = i;
         larger_in_right[i] = n-i-1;
     }
        stack<int> sl,sr;
     for(int i=0;i<n;i++){
         while(!sr.empty() && A[sr.top()] > A[i])
         {
             larger_in_right[sr.top()] = i - sr.top()-1;
             sr.pop();
         }
         sr.push(i);
     }
     for(int i=n-1;i>=0;i--){
         while(!sl.empty() && A[sl.top()] >= A[i])
         {
             larger_in_left[sl.top()] = sl.top()-i-1;
             sl.pop();
         }
        sl.push(i);     
     }  
        int ans = 0;
        for(int i=0;i<n;i++){
            ans += A[i]*(larger_in_left[i] + 1)*(larger_in_right[i]+ 1);
            if(ans > MOD)
                ans -= MOD;
        }
        return ans;
